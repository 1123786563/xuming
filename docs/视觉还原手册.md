这类风格包含了大量的发光特效、故障艺术和非标准控件，开发者非常需要明确的参数规范。以下是为您准备的视觉还原手册核心内容：

1. 基础色彩系统 (Atomic Colors)
The Void (背景色)：#000000 (极致黑/OLED Black)。注：禁止使用灰色渐变，保持纯黑以突出荧光效果。
Life Signal (核心绿)：#00FF41 (Matrix Green)。用于：HP血条、成功反馈、主要按钮。
Nuclear Warning (危机红)：#FF003C (Glitch Red)。用于：低血量警报、核打击提醒、紧急抢修按钮。
Bio-Upgrade (赛博紫)：#BC13FE (Cyber Purple)。用于：商店解锁、高级插件、成就勋章。
Caution Yellow (工业黄)：#F2FF00 (Warning Yellow)。用于：生存评估弹窗、中度压力提醒。
2. 字体与排版规范 (Typography)
标题/数据 (Pixel)：推荐使用开源像素字体如 Zpix 或 DotGothic16。
大标题：32pt / Bold / 字间距 2px。
数据动态：24pt / Monospace (确保数字跳动时不发生位移)。
正文/毒舌文案 (Monospace)：推荐 JetBrains Mono 或 Roboto Mono。
正文：14pt / Medium / 行高 1.5。
装饰文字：10pt / 灰色 #444444。用于角落的 SYS_LOG 等装饰性代码。
3. 特效实现技术指南 (FX Technicals)
发光效果 (Glow)：
禁止使用简单的阴影，建议开发者使用 渲染着色器 (Shaders) 或 多层阴影叠加。
参数建议：BoxShadow(color: color, blurRadius: 10, spreadRadius: 2)。
故障艺术 (Glitch Effect)：
触发时机：点击“续命”按钮或弹出“核打击提醒”时。
实现方式：随机位移像素（Offset）、RGB 分离（Color Channel Split）以及 20ms 的白噪点闪烁。
磨砂玻璃 (Glassmorphism)：
应用位置：场景选择卡片、弹窗背景。
参数：BackdropFilter(sigmaX: 10, sigmaY: 10)，背景色为 #1C1C1E 透明度 70%。
4. 3D 模型渲染规范 (3D Assets)
格式要求：建议使用 .glb 或 .gltf 格式进行轻量化加载。
光照预设：单一顶部冷光（Neon White）+ 侧面边缘光（Rim Light，颜色随 HP 状态切换：绿/红）。
热力图实现：通过 Texture Mask 在模型特定区域（颈椎、腰椎）叠加动态的红色自发光材质（Emission Map）。
5. 核心控件交互规范 (Component UI)
能量槽 (Energy Bar)：必须是“格状”填充，而非平滑进度条。填充时伴随从左至右的呼吸式点亮动效。
拟物按钮：点击时应有明显的“下沉”视觉反馈，并伴随 HapticFeedback.mediumImpact 物理震动。
