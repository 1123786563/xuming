这是一份为您精心编写的 “续命 (XuMing)” App 界面设计与交互逻辑说明文档。本设计将赛博朋克美学（Cyberpunk）与职场久坐健康痛点深度结合，通过“生物监控系统”的隐喻，构建了一个从危机触发到行动修复的完整业务闭环。

一、 设计语言：赛博求生 (Cyber Survival)
配色规范：以 OLED 极致黑 (#000000) 为底色，象征职场高压环境；以 黑客矩阵绿 (#00FF41) 代表生命值与修复；以 警告故障红 (#FF003C) 提示生理危机；以 赛博紫 (#BC13FE) 标记高级功能与系统升级。
字体规范：全站使用 等宽点阵字体 (Monospace Pixel)，模拟精密仪器的数据流感。
视觉元素：采用工业风几何边框、全息 3D 人体模型、以及带有 Glitch（故障艺术）特效的图标。
二、 用户旅程与界面串联逻辑
1. 系统接入与初始化 (Onboarding Phase)
[系统认证-登录页]：作为入口，通过 Logo 的微弱故障闪烁建立品牌调性。用户通过手机或社交账号“注入生命信号”。
[身体系统初始化诊断页]：新用户首屏，通过“生物扫描”交互收集久坐时长等数据，将现实身体状况“数字化”。
[生存现状评估报告弹窗]：诊断后即刻跳出的强交互界面，利用低评分（如 24/100）和红色预警产生“焦虑 Hook”，驱动用户点击“立即续命”进入主循环。
2. 实时监控与危机响应 (The Core Loop)
[生存仪表盘-终极版]：App 的中心枢纽。写实 3D 角色通过动态热力图（颈部/腰部红光）实时展示身体受损点。
[核打击级-久坐提醒页]：当检测到用户 60 分钟未活动时，系统强行弹出最高等级的红色预警界面，辅以“轮椅退休金”等毒舌文案，打破用户的行动拖延。
[极高压力针对性动作推荐]：从提醒页一键跳转，系统根据诊断数据精准匹配“脊椎解压”等抢修任务。
3. 任务执行与感官修复 (Action Phase)
[脊椎解压任务执行页]：写实 3D 指引演示精准动作。
[音效同步-骨骼复位页]：开启 ASMR 模式，通过声波可视化器、模拟“咔哒”声和手机震动，让拉伸运动具备如同“机械校准”般的爽快感。
[压力释放对比报告页]：运动结束后，通过“红转绿”的 3D 模型对比，直观展示 42% 的压力释放，完成正向激励反馈。
4. 职场摸鱼与隐私护航 (Stealth Strategy)
[场景切片选择]：用户根据当前环境切换模式（椅子/站立/墙面）。
[会议潜行协议]：专为会议设计，整合桌下踝泵、腹部收缩等隐蔽动作，低亮色调防止被旁人察觉。
[Boss Mode 伪装界面]：全站最核心的安全开关。用户通过手势（如摇一摇）可将运动界面瞬间切换为 [VS Code 代码界面] 或 [Excel 表格界面]，完美掩盖续命行为。
5. 经济系统与长期增长 (Growth & Social)
[每日生存任务-业务中心]：将运动、社交、商业变现（补丁激励）整合为每日清单，产出“续命币”。
[续命商店]：金币的消耗终端。通过 [系统插件注入动效] 解锁新动作和场景，满足用户的收集与升级欲。
[生存成就-勋章墙]：展示长期荣誉，如“核打击幸存者”，增加用户粘性。
[给同事续命-社交裂变页]：通过向同事投递“毒舌炸弹”实现病毒式传播。
6. 总结与回顾 (Retention)
[上周生存周报弹窗]：每周一通过五维雷达图和扎心总结，让用户对自己的身体“系统存续状态”保持长期关注。
[实时生命值桌面组件]：将 HP 血条和毒舌语录常驻手机桌面，实现全天候的品牌刷脸。
三、 核心交互说明
危机触发：App 检测到静止数据后，主动推送高分贝、强震动的 Level 3 提醒。
极速切换：在任何动作指引页，右下角红色按钮一键进入“Boss Mode”。
数据联动：所有动作产生的压力下降数值，会实时同步至个人档案的 HP 槽。
通过这套串联，“续命”App 不再是一个枯燥的健身工具，而是一个充满趣味、危机感与保护机制的“职场生存 OS”。
